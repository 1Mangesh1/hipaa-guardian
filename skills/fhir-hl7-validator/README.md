# FHIR & HL7 Validator

Healthcare data format validation and compliance checking.

## Quick Start

See [SKILL.md](./SKILL.md) for full documentation and usage examples.

## Key Features

- FHIR R5 resource validation
- HL7 v2 message parsing
- CDA clinical document validation
- Terminology validation (SNOMED CT, LOINC, ICD-10)
- Reference resolution
- Interoperability checking
